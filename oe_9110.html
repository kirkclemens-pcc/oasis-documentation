Edit Text If state law prohibits transmission of certain ICD codes (e.g., HIV or STD diagnoses), these codes will be removed by iQIES if found on records submitted by HHAs within that state. If any item within an ICD list contains such a code on a submitted record, it will be removed from the list. If any of these items is removed, but a valid code remains in a subsequent item on the list, the subsequent codes will be "moved up" so that no missing codes remain.

Therefore, if a non-blank value is required in an ICD code list and all of the submitted items in that list contain blank and/or prohibited codes, a fatal error will result and the record will be rejected. As long as at least one item in such a list contains a non-blank, correctly-formatted, allowable code, a fatal error will not occur.
