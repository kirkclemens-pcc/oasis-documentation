Edit Text In order to modify or inactivate a record that has previously been accepted by the system, the system must be able to locate the previous record. Appropriate values from the record to be corrected must therefore be submitted for the following locator items:

M0030_START_CARE_DT
M0032_ROC_DT
M0032_ROC_DT_NA
M0040_PAT_FNAME
M0040_PAT_LNAME
M0064_SSN
M0064_SSN_UK
M0066_PAT_BIRTH_DT
M0069_PAT_GENDER
M0090_INFO_COMPLETED_DT
M0100_ASSMT_REASON
M0906_DC_TRAN_DTH_DT

If a matching previously accepted record cannot be located, a fatal error will result and the submitted record will be rejected.
